# TVmaze PHP

Este projeto é uma aplicação web desenvolvida em PHP para interagir com a API do TVmaze e exibir informações sobre programas de TV.

## Descrição

**TVmaze PHP** é uma aplicação que permite aos usuários buscar informações sobre programas de TV utilizando a API do TVmaze. A aplicação oferece uma interface simples para pesquisa e exibição de detalhes dos programas, como elenco, episódios e resumos.

## Tecnologias Utilizadas

- **PHP**: Para o backend e lógica do servidor.
- **HTML**: Estrutura das páginas web.
- **CSS**: Estilização e layout.
- **JavaScript**: Interatividade e manipulação do DOM.
- **API TVmaze**: Para obter dados sobre programas de TV.

## Funcionalidades

- Busca de programas de TV por nome.
- Exibição de detalhes dos programas, incluindo elenco e episódios.
- Interface amigável e responsiva.

## Instalação

1. Clone este repositório para o seu ambiente local:
    ```bash
    git clone https://github.com/AndersonC96/TVmaze_PHP.git
    ```
2. Navegue até o diretório do projeto:
    ```bash
    cd TVmaze_PHP
    ```
3. Configure o servidor PHP para servir os arquivos do projeto.

4. Abra o projeto em seu navegador:
    ```arduino
    http://localhost/TVmaze_PHP
    ```

## Uso

1. Navegue até a interface principal da aplicação.
2. Utilize a barra de pesquisa para buscar programas de TV pelo nome.
3. Clique em um programa nos resultados da pesquisa para ver os detalhes, incluindo elenco e episódios.
